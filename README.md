## üöÄ How to Run
To run any of these files locally, ensure you have a C++ compiler installed (like `g++`).

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/stealthcamaro/CPlusPlus-Basics.git](https://github.com/stealthcamaro/CPlusPlus-Basics.git)
    cd CPlusPlus-Basics
    ```

2.  **Compile a file (example):**
    ```bash
    g++ linkedlist.cpp -o my_program
    ```

3.  **Run the executable:**
    ```bash
    ./my_program
    ```
